# smps-reverse-engineering

Repository containing reverse-engineered SMPS schematics and PCB teardown images from real-world AC-DC power supplies.

This project documents commercial SMPS units for educational purposes and to support independent repair and hardware study.

> [!NOTE]
> This repository does not provide repair guarantees, modification guides, or safety certification.

## Overview
This repository contains:
- Reverse-engineered schematic redraws

- PCB teardown images

- Component reference mapping (where applicable)

No proprietary firmware, copyrighted documents, or trade secrets are included.

The purpose of this repository is documentation, study, and repair facilitation.
